# Surgery Timing and 30-Day Mortality Outcomes
**Author:** Yanqi Huang
**Report:** [view analysis report](2025-12-02_report-only_surgery-timing-outcome_yanqi-huang.pdf) 

## Overview

This project investigates whether the **hour of day a surgery is performed** is associated with **30-day postoperative mortality**, after adjusting for patient demographics, clinical risk factors, comorbidities, and procedure type. The analysis uses logistic regression with restricted cubic splines to flexibly model non-linear relationships.

## Analysis Pipeline

The analysis is organized into four sequential R scripts:

### Step 0 — Setup (`0.0_setup_surgery-timing-outcome.R`)
- Loads all required R packages (auto-installs missing ones)
- Reads `Surgery_Timing.csv`
- Defines variable groupings (exposure, outcomes, covariates)
- Converts columns to appropriate types (binary, categorical, continuous)

### Step 1 — Exploratory Data Analysis (`0.1_EDA_surgery-timing-outcome.R`)
- Summarizes data structure and missingness (overall and by outcome group)
- Produces histograms, boxplots, density plots, and bar charts for all variables
- Plots crude 30-day mortality rates by surgery hour (binned and LOESS-smoothed)
- Runs chi-square tests for categorical variable associations with `mort30`
- Generates correlation matrix of continuous predictors

### Step 2 — Data Processing (`0.2_process-data-to-model_surgery-timing-outcome.R`)
- Restricts to adult patients (age ≥ 18; ~212 rows removed)
- Assesses multicollinearity via VIF and correlation; drops `bmi`, `race`, `ccsComplicationRate`, `complication_rsi`
- Collapses 27 specific `ahrq_ccs` procedure codes into 8 grouped categories (Gastrointestinal & Hernia, Breast, Orthopedic & Spine, Gynecologic, Urologic, Endocrine, Plastic, Other)
- Bins `hour` into six time-of-day categories (6–8am, 8–11am [reference], 11am–1pm, 1–3pm, 3–5pm, 5–7pm)
- Produces a baseline characteristics table stratified by hour category (`gtsummary`)
- Saves cleaned dataset to `data/processed/` as an RDS file

### Step 3 — Model Selection & Results (`1.0_select_model_surgery-timing-outcome.R`)
- Fits logistic regression models (`lrm` from the `rms` package) comparing:
  - Linear terms vs. restricted cubic splines (RCS) with K = 3, 4, 5 knots for `age` and `mortality_rsi`
- Selects best-fitting model: linear `age`, RCS(3 knots) for `mortality_rsi`, categorical `hour`
- Runs outlier and influence diagnostics (Cook's D, leverage, DFBETAS, DFFITS, jackknife residuals)
- Generates odds ratio forest plot and a comparison of unadjusted vs. adjusted hour effects

### Report (`2025-12-02_rmd-for-report.Rmd`)
- Self-contained R Markdown document that renders a full PDF report
- Integrates all tables, figures, and model output with written interpretation
- Includes a code appendix

## How to Run

1. **Clone the repository** and open the project in R or RStudio.

2. **Update file paths.** The scripts use absolute paths; update the working directory variable at the top of each script to point to your local clone.

3. **Run scripts in order:**

   ```r
   source("code/0.0_setup_surgery-timing-outcome.R")   # Load packages & data
   source("code/0.1_EDA_surgery-timing-outcome.R")      # Exploratory analysis
   source("code/0.2_process-data-to-model_surgery-timing-outcome.R")  # Data prep
   source("code/1.0_select_model_surgery-timing-outcome.R")           # Modeling
   ```

4. **Render the report:**

   ```r
   rmarkdown::render("code/2025-12-02_rmd-for-report.Rmd")
   ```

**Expected outputs:**
- `data/processed/` — cleaned RDS dataset
- `results/2025-12-02/` — PNG figures (baseline table, univariable summary, OR forest plot, hour effect comparison, RSI effect plot)
- A rendered PDF report

## Key Findings

- The final logistic regression model included `age` (linear), `mortality_rsi` (RCS, 3 knots), `hour_cat` (6 time-of-day bins), `gender`, `asa_status`, `procedure` group, `baseline_charlson`, and 8 binary comorbidity indicators.
- `mortality_rsi` showed a strong, non-linear positive association with 30-day mortality.
- After adjustment, the effect of surgery hour on 30-day mortality was attenuated compared to the unadjusted analysis, suggesting that case-mix (higher-risk patients scheduled at certain times) partially explains crude hour-of-day differences.
